Lesson 3: Working with Colors in CSS

Lesson 3: Working with Colors

  • Color Names:
    CSS allows you to use predefined color names like red, blue, and green for styling. This is an easy way to apply colors without needing to remember codes.

  • HEX Color Codes:
    HEX codes represent colors using a six-character code, starting with a #. For example, #FF5733 is a shade of red. Each pair of characters represents the red, green, and blue components of the color.

  • RGB Values:
    The RGB color model is defined using three values for red, green, and blue components, each ranging from 0 to 255. For example, rgb(255, 87, 51) is the same red color as the HEX code above.

  • HSL Values:
    HSL stands for Hue, Saturation, and Lightness. It allows for more intuitive color adjustments. For example, hsl(9, 100%, 60%) gives you a shade of red, similar to the previous examples.

    Practical Exercise:

    1. Use Color Names

    1. Set the background color of an element to blue:
      • background-color: blue;

    2. Use HEX Color Codes

    1. Set the text color to a shade of red using a HEX code:
      • color: #FF5733;

    3. Use RGB Values

    1. Set the border color to a green shade using RGB values:
      • border-color: rgb(0, 255, 0);

    4. Use HSL Values

    1. Set the background color to a light blue using HSL:
      • background-color: hsl(200, 100%, 70%);
