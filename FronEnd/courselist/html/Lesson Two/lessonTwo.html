Structuring a Web Page

Lesson 2: Structuring a Web Page

  • Semantic HTML Elements:
    Semantic elements provide meaning and help organize content clearly. Key semantic elements include:

    • <header>: Represents introductory content or navigation links.
    • <nav>: Defines a navigation area, often containing links.
    • <section>: Groups related content.
    • <article>: Self-contained content, such as blog posts or news stories.
    • <footer>: Contains footer information like copyright, contact, etc.
    • <aside>: Content related to the main content, like sidebars.

  • Organizing Content:
    Structuring a web page involves grouping related content using these elements to improve accessibility and readability.

    Practical Exercise:

    1. Create a new HTML file.
    2. Add a semantic structure with the following:
      • A header containing the website's title.
      • A navigation bar with links.
      • A main content section with two articles.
      • A sidebar (aside) with related links.
      • A footer with copyright information.
    3. Validate the structure using an HTML validator tool.

    Summary:

    • Use semantic elements to organize and structure your HTML document.
    • Semantic tags improve both accessibility and search engine optimization (SEO).
    • Key tags include: <header>, <nav>, <section>, <article>, <aside>, and <footer>.
